🚀 What's New in v1.0

✅ Initial public release of our free and open-source video object removal tool!
This version includes all the essential features needed to perform brush-based object removal from videos with frame-level precision.

---

✨ Key Features

- 🎯 Frame-by-frame navigation and preview
- 🖌️ Adjustable brush-based masking with zoom support
- 🔁 Two masking modes: "Single Mask" and "Per-Frame Mask"
- 🧹 OpenCV-based inpainting for object removal
- 🎞️ Rebuild video using FFmpeg with original frame rate
- 🧭 Easy navigation: Previous/Next frame buttons
- 💾 Export cleaned video as `.mp4`
- ▶️ One-click preview of rebuilt video
- 📝 Real-time log panel + status bar
- 🌙 Dark mode professional GUI built with Tkinter
- 🧑‍💻 Precompiled Windows EXE for non-technical users
- 🌐 100% free and open-source under MIT License

---

🛠 Technical Stack

- Python 3.10+
- Tkinter (GUI)
- OpenCV (object removal)
- FFmpeg (video extraction & rebuild)
- Pillow (image rendering)
- Threading (non-blocking operations)
